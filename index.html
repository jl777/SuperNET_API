SuperNET API

SuperNET enables people that develop with virtual currencies be able to connect to several Blockchain unitilities with one Software.

It is also designed to help supporter, user and fans of peer-to-peer cryptographic technology to create content and distribute information on the network.The technology is designed to be integrated directly into web devices, finally making legal digital content distribution a first-class citizen on the web.

SuperNET is also a reference implementation of the Nxt and BitcoinDark (SuperNET) protocol.

The SuperNET web platform is an open standard that enables web browsers and web devices to perform micropayments and copyright-aware, peer-to-peer digital media distribution.
